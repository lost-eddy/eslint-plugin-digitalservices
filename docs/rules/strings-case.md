#Правила оформления строковых констант
>**Strings formatting rules**

Все ключи должны быть в регистре **camelCase**

>All keys must be in **camelCase**

Пример:
>Example:
```js
const strings = {
  hello: "hello",
  base64: "base64",
  headerBlock: {
    authorizationKey: "Authorization" 
  }
}
```
